# 河北移动 · 社会渠道酬金计算器

> Commission Calculator for China Mobile Hebei — Social Channel (incl. China Tietong)

一个面向河北移动社会渠道（含铁通）的酬金计算工具，覆盖酬金框架 + 手续费框架全部业务场景。**单文件 HTML，无需安装，浏览器直接打开即可使用，支持离线运行。**

---

## ✨ 功能特性

### 💰 酬金计算
- 预置 **41 条酬金规则**，覆盖主套餐（新入网/升档/全家福）、非主套餐（宽带/电视/连续包月/趸交/话费合约等）、手续费（拉新/转化/融合/终端等）全场景
- 全局填写**客户套餐信息**，宽带、电视、分期金币等业务自动继承套餐贡献，无需重复输入
- 支持新入网 / 升档 / 存量加装三种场景，升档差价自动计算
- **叠加项一键勾选**（冀享、新身份证、关怀、卡槽运营·重点客群、细分/竞对/收费副卡等），实时计算并显示完整公式

### 📋 历史记录
- 保存每笔计算记录（客户手机号、姓名、日期、备注、全部业务明细）
- 按**手机号搜索**、**日期范围筛选**
- 点击展开查看酬金详情（每项业务的计算公式和金额）
- 支持**复用**（一键载入计算器重新编辑）
- **导出 CSV**（Excel 直接打开，支持中文）

### 📊 月度汇总看板
- 按月统计：总酬金、办理笔数、笔均酬金、客户数
- 分类占比（主套餐 / 非主套餐 / 手续费）柱状图
- 业务酬金排行 TOP10

### 🖨 单据打印
- 生成简洁的酬金计算单据
- 支持打印和截图，方便贴台账或分享

### 📦 快捷模板
- 保存常用业务组合为模板（如「新入网99+宽带+电视」），自定义命名
- 一键载入模板，自动填充套餐贡献和业务列表

### ⚙️ 规则库管理
- 可视化增删改全部酬金规则
- 支持「比例×基数」和「固定金额×数量」两种计算方式
- 自定义叠加项及其比例
- 规则导出/导入（JSON 格式），团队共享
- 一键恢复默认规则

## 📖 使用指南

### 基本流程

```
填写客户信息（手机号/姓名/日期）
    ↓
填写套餐贡献（新入网填当前值，升档两侧都填）
    ↓
选择业务 → 添加 → 填写参数 / 勾选叠加项
    ↓
查看酬金汇总 → 保存历史 / 打印单据
```

### 三种业务场景

| 场景 | 套餐贡献填法 | 说明 |
|:---|:---|:---|
| **新入网** | 只填「当前套餐贡献」 | 如新入网 99 元套餐，填 99 |
| **升档** | 左右都填 | 如 59→99，左填 99、右填 59，差价自动算为 40 |
| **存量加装** | 只填「当前套餐贡献」 | 如老客户 59 元套餐加装宽带，填 59 |

### 计算示例

**新入网 99 元（冀享 + 新身份证 + 关怀 + 重点客群）+ 宽带 + 电视：**

| 业务 | 公式 | 酬金 |
|:---|:---|---:|
| 新入网-99元及以上 | (400%+200%+200%+400%+50%) × 99 × 1 | ¥1,237.50 |
| 宽带+电视 | 200% × 99 × 1 | ¥198.00 |
| **合计** | | **¥1,435.50** |

## 🏗 技术方案

- **单文件架构**：HTML + CSS + JavaScript，零依赖，零构建
- **数据存储**：浏览器 localStorage（规则、历史、模板各自独立存储 key）
- **离线可用**：不依赖任何外部 CDN 或 API
- **导入导出**：规则 JSON + 历史 CSV，支持跨设备迁移
- **兼容性**：Chrome / Edge / Firefox / Safari 主流浏览器均可使用

### 存储结构

| Key | 内容 | 说明 |
|:---|:---|:---|
| `hbyd_rules_v3` | 酬金规则 | JSON 数组，41 条默认规则 |
| `hbyd_hist_v1` | 历史记录 | 每条含完整业务明细和计算结果 |
| `hbyd_tpl_v1` | 快捷模板 | 含套餐信息 + 业务组合 |

## ⚠️ 注意事项

- 数据存储在**本机浏览器**中，清除浏览器数据会导致历史记录丢失，建议定期导出备份
- 规则口径以**河北移动正式酬金文件**为准，本工具仅作计算辅助
- 不同浏览器之间数据不互通，可通过「导出规则」→「导入规则」迁移
